PDH Recommendation App

A community-powered meal review system for Panther Dining Hall at Florida Tech.

Overview

The PDH Recommendation App was built to give students at Florida Tech a voice in what meals are served at Panther Dining Hall. While meal recommendations are a future feature, the app currently allows users to:

  • Review meals currently being served at PDH
  • Browse or search meal reviews via a clean, interactive dashboard
  • Write and vote on meal suggestions for PDH staff consideration

Technical Details

  • Frontend: Flutter framework using Dart
  • Backend: Python script for daily menu scraping from PDH sources
  • Database: Firebase Firestore for data storage
  • Authentication: Firebase Authentication for secure login
  • Image Hosting: Firebase Storage for meal and suggestion images
